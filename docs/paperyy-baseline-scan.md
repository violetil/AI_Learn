# PaperYY 向：中文正文基线扫描（第二轮前）

## 扫描范围
- 文件：`thesis.tex`
- 区间：自 `\chapter{绪论}` 起至 `\backmatter` 止（**不含摘要/Abstract**）

## 高频特征统计（初扫）
| 模式 | 出现次数（约） | 说明 |
|------|----------------|------|
| `本文` | 29 | 作者主语密度偏高，易被判定为模板化叙述 |
| `系统` | 62 | 合理技术词，但与泛化句连用时会放大“概括腔” |
| `因此` | 5 | 转折模式之一 |
| `同时` | 4 | 转折模式之一 |
| `一是/二是/三是` | 各 2 | 典型列举模板，绪论等处仍存在 |
| `可见` | 2 | 结论引导词 |

## 优先改写段落（人工标注）
1. **第1章 `\section{研究背景}`**：首段“一是…二是…三是…”结构。
2. **第2章 `\section{生成式 AI 教育应用研究}`**：综述段句式接近摘要并列。
3. **第3章 `\section{关键评价指标}`**：多处以“本文…”起句，与其他章节奏雷同。
4. **第5章**：流程描述缺少与仓库路径的一一对应（本轮补 `\path{...}` 锚点）。
5. **第6章**：解释句可与采样脚本、日志路径绑定（已有 `docs/...`，补强表述）。
6. **第7章**：总结段仍偏对称 enumeration，需打散句首模式。

## 本轮策略摘要
- 降“本文”密度：改为无主语句、被动表述或“本课题/实现上”等交替。
- 插入**证据映射内已有路径**（见 `docs/thesis-evidence-map.md`），避免虚构模块。
- 保留公式、表值、API 路径字符串与 `\cite`/`\bibitem` 映射不变。
